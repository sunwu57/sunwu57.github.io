    Code 
    int found(int a[], int left, int right, int x) {
    while (left < right) {
        int mid = (right + left) >> 1;
        if (a[mid] < x) left = mid + 1;
        else
            right = mid;
    }
    return left;
}

    对于 <>的特殊符号应该用 < > 代替，不然会显示错误。(只需要修改 < 替换为 < 即可）

    必须紧贴代码开头（避免多出首行），需单独一行（避免少了尾行）
